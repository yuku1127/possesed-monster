----------操作説明------------

キーボードの操作割り当ては以下の通りです。  
D:攻撃  A:ヒョーイ/ヒョーイ解除（遠距離攻撃）  W:インベントリを開く  Space:ジャンプ
  
PCの音量を最小にしておくことを推奨します。
ゲーム画面の大きさは開始する前のブラウザのウィンドウサイズに設定されるため、あらかじめフルスクリーンにしておくことを推奨します。
またゲーム終了はブラウザを閉じる、または再読み込みで終了します

本作品に使用されている一部の素材は、以下の提供元より使用許諾を得て利用しています
サウンド: MOMIZizm MUSiC　、FC音工場

----------仕様解説------------

コードの大まかな仕様
{
---main.jsの役割---

全てのスクリプトの処理を最終的にはこのmain.jsで毎フレーム実行する。各クラス共通の変数をまとめて定義し、各クラスに一つだけ必要なインスタンスはmain.js実行時に一度だけ生成(行番号81~92)、それ以外のクラス（行番号72~）はmap.js(次の項目参照)のタイルを識別して複製する。
メインループ関数（行番号230~）の中でupdate関数(入力された内容からデータ更新)とdraw関数（更新された値を描画に反映）をGAME_FPSの周期で繰り返し呼び出す
プレイヤーの入力はイベントハンドラで管理し各クラス共通の変数値を変更する

--—map.jsの役割—--

16*16ピクセルのタイル約７０種類をTiledというマップエディタを使ってステージに配置し、それを各タイルに番号をつけたfieldData配列に変換[(map.js行番号１〜より)
プレイヤーの座標から仮想キャンバス範囲内のタイルを全て描画（map.js行番号197のdraw関数）、特定のタイルからはモンスターやギミックのあるオブジェクトを生成


}
