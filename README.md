こちらはGithubで公開している『PossessedMonster』の操作、補足説明になります。  
ゲーム内での説明が不十分になってしまったためこちらで補足させていただきます。  
キーボードの操作割り当ては以下の通りです。  

D:攻撃  A:ヒョーイ/ヒョーイ解除（遠距離攻撃）  W:インベントリを開く  Space:ジャンプ
  
デフォルトの音量が大きくなってしまったため、PCの音量を最小にしておくことを推奨します。
ゲーム画面の大きさは開始する前のブラウザのウィンドウサイズに設定されるため、あらかじめフルスクリーンにしておくことを推奨します。
またゲーム終了はゲーム内の仕様には組み込まれてないため、ブラウザを閉じる、または再読み込みで終了します
ゲームリセットはホーム画面でshiftを押すことで行えます。
